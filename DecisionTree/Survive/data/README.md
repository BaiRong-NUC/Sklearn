# Titanic 数据集字段说明

本文件用于说明 [data.csv](data.csv) 中各字段的业务含义、数据类型与建模用途。

## 1. 数据集简介

该数据集记录了泰坦尼克号乘客信息，用于分析和预测乘客是否幸存。

- 任务类型：二分类
- 预测目标：Survived
- 目标含义：0 表示未幸存，1 表示幸存

## 2. 字段说明

| 变量名      | 含义         | 类型              | 说明                                                                   |
| ----------- | ------------ | ----------------- | ---------------------------------------------------------------------- |
| PassengerId | 乘客编号     | 标识符            | 乘客唯一 ID，通常不直接作为预测特征                                    |
| Survived    | 幸存状态     | 分类（0/1）       | 预测目标（Target）                                                     |
| Pclass      | 船票等级     | 有序分类（1,2,3） | 1=头等舱，2=二等舱，3=三等舱，反映社会经济地位                         |
| Name        | 姓名         | 文本              | 乘客姓名，可能包含称谓等信息                                           |
| Sex         | 性别         | 分类              | male=男，female=女                                                     |
| Age         | 年龄         | 数值              | 乘客年龄，存在缺失值                                                   |
| SibSp       | 亲友人数     | 数值              | 同船的兄弟姐妹/配偶数量                                                |
| Parch       | 父母子女人数 | 数值              | 同船的父母/子女数量                                                    |
| Ticket      | 船票号码     | 文本              | 船票编号                                                               |
| Fare        | 票价         | 数值              | 乘客票价                                                               |
| Cabin       | 客舱号       | 文本              | 客舱编号，缺失值较多                                                   |
| Embarked    | 登船港口     | 分类              | C=Cherbourg（瑟堡），Q=Queenstown（皇后镇），S=Southampton（南安普敦） |

## 3. 建模提示

- 建议将 Survived 作为标签列，其他字段作为候选特征。
- 文本字段 Name、Ticket、Cabin 需要做特征提取或编码后再使用。
- Age、Cabin、Embarked 等字段需先处理缺失值。
- Pclass 是有序分类变量，可保留其顺序信息。
